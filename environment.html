Chart using data from OWID, by Charlie Meyrick

Chart notes

This chart pulls live pollution data from emissions-api.org. The data are produced from satellite imagary. The API works nicely and has a (relatively) intuitive interface that allows you to build your own API url to make the call.

The visulisation is made in Vega Lite. Data is pulled from the API twice, once requesting data for the UK, and once for Bristol. This creates two charts, which are superimposed on the same axes using the Layer command.

The JSON code that defines the chart is available at the RapidCharts GitHub repository, here.

Richard Davies, July 2021
