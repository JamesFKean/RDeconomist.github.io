Covid-19

This is a site I built late at night while thinking about how to improve the way we present data at the Economics Observatory. We have just lived through the most data-rich crisis ever, and the graphical analysis of it has been weak. Despite the availability of official daily data, most of what we see is days or weeks out of date. The charts here are all live, either running direct from the original data sources, or are updated daily via a quick script written by me. All off the code — HTML, CSS and JavaScript — is by me and I am expert in none of this, so please do suggest improvements. The charts are drawn in Vega-Lite, which has an excellent support site here. I also used a number of tutorials avaiable on the Observable website. You can find this project on GitHub. Please play fork and improve on it

Richard Davies
