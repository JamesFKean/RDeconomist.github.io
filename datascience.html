Week 1 - Computer lab

Building an interactive web site

In this course we are going to be extracting data from web sites that others have built, including via web scraping. To prepare for that, it is helpful to understand the basics of how the web works. In our first computer lab we learn the basics of HTML, CSS and Javascript as you build your first web site. We also learn how to embed our first automated and interactive chart. Over the next 10 weeks this site will become your coursework submission.
